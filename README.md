# 🍽️ Meal Finder

A simple JS app to:

- 🔍 Search meals by name  
- 🎲 Get a random recipe  
- 📋 View ingredients & instructions  

Built with **HTML**, **CSS**, **JS (Fetch API)** + [TheMealDB](https://www.themealdb.com/api.php)
